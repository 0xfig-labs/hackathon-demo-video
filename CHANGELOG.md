# Changelog

## 0.1.0 - 2026-06-19

### Features
- Initial release of hackathon-demo-video skill
- Playwright browser recording pipeline with warmup caching
- edge-tts Chinese/English narration with WebVTT subtitles
- Pillow subtitle overlay rendering with CJK font support
- ffmpeg composition with intro frame, layout modes, and audio mixing
- Multiple output formats: landscape, portrait, square
- Multiple capture modes: desktop, mobile, tablet
- Layout options: native, fit, crop, framed
- Custom interaction steps: click, fill, link, wait_text, scroll, wait
- Adjustable TTS voice, speed rate, and ffmpeg encoding presets
- Per-run isolated working directories for parallel execution
- Non-strict mode for exploratory step execution
