# 旁白脚本写作指南

## 核心原则：旁白跟着画面走

❌ **不要写 README 风格（这是文档，不是旁白）：**
```
TradeTrace 是一个基于 Sui 的 DeepBook Predict 结果验证工具。
在 DeFi 预测市场中，用户创建头寸后需要验证结算是否准确。
```

✅ **要写 VO 风格（就像你在给评委讲解）：**
```
欢迎使用 TradeTrace。今天我们来验证一笔预测交易。
首先把交易编号粘贴到输入框，点击分析。
分析结果出来了——这里展示了头寸类型、方向和执行价。
```

## 时间估算

| 语速 | 每字时长 | 例：100 字 |
|------|----------|-----------|
| 正常 `+0%` | ~4 字/秒 | ~25s |
| 慢速 `-5%` | ~3.5 字/秒 | ~28s |

短 demo 推荐 **60-90s**，旁白约 **200-350 字**；正式黑客松提交推荐 **4–5 分钟**，旁白约 **700–1000 英文词** 或 **900–1400 中文字**。

## 节奏模板（60-90s）

```
0-10s    开场介绍 + 页面展示
10-25s   操作步骤 + 加载等待
25-45s   核心功能讲解（对应画面内容）
45-65s   特性/亮点展示（证据/存储/开源）
65-75s   结尾 + 价值总结
```

## 语音与语速

```bash
# 可用中文男声
--voice zh-CN-YunyangNeural   # 专业可靠（推荐）
--voice zh-CN-YunjianNeural   # 激情

# 中文女声
--voice zh-CN-XiaoxiaoNeural  # 温暖
--voice zh-CN-YunxiNeural     # 阳光

# 查看所有中文语音
edge-tts --list-voices | grep zh

# 语速调整
--rate "+0%"   # 正常（默认）
--rate "-5%"   # 慢速（推荐长视频）
--rate "+8%"   # 快速（视频短于旁白时用）
```
